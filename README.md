# Digital Library Management System / Sistemi për Menaxhimin e Bibliotekës

An interactive, responsive, and feature-rich web application designed to manage a library's book inventory, track student borrowings, and visualize library statistics in real-time. Built using semantic HTML5, modern CSS3 (with dynamic variables and theme switching), and vanilla JavaScript.

## 🚀 Features

### 1. Dashboard & Real-Time Statistics
* **Dynamic Overview Counters:** Displays total books, currently borrowed books, active members, and overdue warnings.
* **Visual Charts:** Includes built-in analytical charts tracking monthly registration trends and book categories using Chart.js.

### 2. Inventory Management
* **Book Registry:** Form to add new books with specific details (Title, Author, ISBN, Category, Publication Year, Copies available).
* **Advanced Search & Filter:** Real-time filtering of books by title/author and category dropdowns.
* **Status Indicators:** Color-coded badges indicating availability status (`Në dispozicion` or `E rezervuar`).

### 3. Borrowing & Circulation Tracking
* **Active Loans Table:** Tracks which student borrowed which book, including issue dates and return deadlines.
* **Actionable Workflows:** Simple button actions to mark books as returned directly from the interface.

### 4. UI/UX Features
* **Theme Toggle:** Full support for **Light Mode** and **Dark Mode** layouts to enhance readability.
* **Responsive Layout:** Fully optimized for desktop, tablet, and mobile displays using a sleek sidebar navigation design.
* **Alert Notifications:** Integrated clean toast messages for validation errors and successful actions.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox/Grid, Custom Variables)
* **Scripting:** Vanilla JavaScript (ES6+)
* **Libraries:** [Chart.js](https://www.chartjs.org/) (via CDN) for data visualization
* **Icons:** [Font Awesome](https://fontawesome.com/) for interface iconography
